## Overview
This project focuses on **scraping, cleaning, and analyzing job postings data from Wuzzuf** to gain insights into the job market.  
It covers the complete data pipeline starting from **web scraping** to **data analysis and visualization**.

---

## Objectives
- Collect job postings data from Wuzzuf automatically  
- Clean and preprocess raw data for analysis  
- Perform Exploratory Data Analysis (EDA)  
- Identify trends in job titles, locations, companies, and required skills  
- Visualize insights to support data-driven decisions
